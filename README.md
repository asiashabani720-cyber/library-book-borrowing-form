Simple PHP + MySQL Library System

Setup
- Import the database schema: `db/schema.sql` into MySQL (via phpMyAdmin or `mysql` CLI).
- Place the project files into your XAMPP `htdocs` folder.
- Edit `config.php` if your DB credentials differ.
-- Open `http://localhost/LIBRARY_SYSTEM/` in your browser.

Notes
- Default admin credentials: username `admin`, password `admin` (change after first login).
